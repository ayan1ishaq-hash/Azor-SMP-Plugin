# Azor Plugin

**Azor Plugin** is a private Minecraft Paper SMP plugin with custom weapons, editable crafting recipes, and in-game recipe GUIs.

> This public repository is intentionally limited. It contains public documentation, license information, issue templates, and release placeholders only. The private source code is not included.

## Features

- Custom SMP weapons
- Custom model data support
- Recipe viewer GUI
- Admin recipe editor GUI
- Per-weapon editable recipe files
- Action bar cooldowns
- Paper server support

## Commands

```text
/recipes
/recipes edit
/recipes give <weapon> [player]
```

## Weapons
