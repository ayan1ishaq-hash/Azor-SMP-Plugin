# Installation

1. Download the latest compiled Azor Plugin `.jar` from the release page.
2. Stop your Paper server.
3. Put the jar into your server's `plugins` folder.
4. Start the server.
5. Edit generated configuration files if needed.
6. Restart the server after major config changes.

## Requirements

- Paper server
- Minecraft/Paper 1.21.x target
- Java 21+

## Folder Created on Server

```text
plugins/AzorPlugin/
```

Weapon recipe files are generated inside:

```text
plugins/AzorPlugin/weapons/<weapon_id>/recipe.yml
```
