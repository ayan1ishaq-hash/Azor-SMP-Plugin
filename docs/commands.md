# Commands

## Player Commands

```text
/az recipes
/recipes
```

Opens the custom weapon recipe viewer GUI.

## Admin Commands

```text
/az edit
/recipes edit
```

Opens the recipe editor GUI.

```text
/az give <weapon> [player]
/recipes give <weapon> [player]
```

Gives a custom weapon to yourself or another player.

## Info Command

```text
/az info <weapon>
```

Shows public item information.

## Craft Limit Command

```text
/az limit <weapon> <number|-1>
```

Sets the max amount of times a weapon can be crafted. Use `-1` for unlimited.
