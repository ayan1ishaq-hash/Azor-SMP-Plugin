# Weapons Overview

This is a public overview only. Exact internal code and implementation details are private.

## Scythe

Role: strongest boss/player killer and group control weapon.

- Soul Dominion
- Reaper's Rift
- Death Sentence

## Katana

Role: fast combo weapon with mobility and AoE slashes.

- Swift Slash
- Dash Cut
- Blade Storm

## Jester's Grin

Role: unpredictable clown sword with execution pressure.

- Chaotic Luck
- Balloon Bomb
- Final Joke

## Demon Sword

Role: demonic damage-over-time sword with Blood Pact power.

- Demonic Hunger
- Hell Chains
- Blood Pact

## Champion's Blade

Role: reliable PvP sword focused on shield pressure and defense.

- Perfect Edge
- Guard Break
- Warrior's Resolve

## Ice Bow

Role: control bow that freezes enemies using normal arrows.

- Frostbite Arrows
- Ice Barrage
- Glacial Prison

## Yumi of the Silent Samurai

Role: precision sniper bow rewarding fully charged shots.

- Perfect Draw
- Piercing Arrow
- Zen Focus
