# FAQ

## Is the source code public?

No. This public folder is designed for documentation and releases only.

## Can I copy or modify the plugin?

No, not without explicit written permission from the owner.

## Do normal arrows work with the bows?

Yes. The bow weapons use normal vanilla arrows.

## Can admins edit recipes?

Yes, using:

```text
/recipes edit
```
